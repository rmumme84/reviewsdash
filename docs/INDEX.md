# acker-site — Índice de Documentos

Use este arquivo como “onde eu procuro o quê”.

- `README.md` — visão geral rápida + links úteis.
- `RUNBOOK.md` — operação do serviço (start/stop/logs/healthcheck), config resumida, troubleshooting.
- `DEBUG.md` — comandos copy/paste para validação e debug por script (sem vazar tokens).
- `CONFIG.md` — lista de ENV vars e arquivos de keys esperados (sem valores) + notas de permissões.
- `GEMINI.md` — uso do Gemini CLI no VPS (agente alternativo).
- `SECURITY.md` — notas sobre hardcoded secrets e recomendações de segurança.
- `CHECKS.md` — checagens rápidas de consistência das docs (ex.: refs `docs/`).
- `CHANGELOG.md` — histórico de mudanças na pasta `docs/`.
- `../CLAUDE_CODE_INTEGRATION.md` — uso do Claude Code no VPS.
